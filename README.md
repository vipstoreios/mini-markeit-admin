# Mini markeit Admin Dashboard - Redesign

ئەم پاکێجە لەسەر فایلە کاراکانی خۆت دروست کراوە. ئەمە دیمۆی جیاواز نییە.

فایلەکان:
- `index.html` — هەمان داشبۆرد، بە layout ـی جوانتر و mobile-first.
- `style.css` — دیزاینی نوێ بە شێوەی سپی/سەوز، کارت، navigation و responsive.
- `script.js` — هەمان JavaScript/Supabase ـی تۆ، بۆ ئەوەی کارکردنەکان نەشکێن.
- `supabase-admin-setup.sql` — SQL ـی ئەدمینەکەت، وەک خۆی.

چۆن بخەیتە GitHub:
1. ئەم ZIP ـە unzip بکە.
2. لە repository ـی `mini-markeit-admin`، ئەم فایلانە replace بکە:
   - `index.html`
   - `style.css`
   - `script.js`
3. `supabase-admin-setup.sql` تەنها ئەگەر پێویستت پێی هەیە دابنێ، ئەگەر لە repo ـکەتدا هەیە replace ـی بکە.
4. چەند خولەکێک چاوەڕێ بکە تا GitHub Pages نوێ ببێتەوە.

تێبینی:
- Supabase URL/anon key و logic ـەکانی `script.js` نەگۆڕدراون.
- ID ـە گرنگەکانی HTML و button onclick ـەکان پارێزراون.
- گۆڕانکارییەکە سەرەکی لە دیزاین و ڕێکخستنەکەیە.
